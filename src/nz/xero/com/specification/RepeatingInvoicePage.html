Feature: Creating New Repeating Invoice

Acceptance Criteria

  1. I can create a repeating invoice with an end date.
  2. I can create a repeating invoice without an end date.
  3. I can create a repeating invoice that repeats every week.
  4. I can create a repeating invoice that repeats every month.

Example 1: I can create a repeating invoice with an end date.

Given:
I create a new repeating invoice with an end date 20 Oct 2020
Then:
It will create a repeating invoice with the end date 20 Oct 2020

Example 2: I can create a repeating invoice without an end date.

Given:
I create a new repeating invoice without an end date
Then:
It will create a repeating invoice displaying no end date.

Example 3: I can create a new repeating invoice that repeats every week.

Given:
I create a new repeating invoice that repeats every week
Then:
It will create a repeating invoice displaying that it repeats every week.

Example 4: I can create a repeating invoice that repeats every month.

Given:
I create a new repeating invoice that repeats every month
Then:
It will create a repeating invoice displaying that it repeats every month.
