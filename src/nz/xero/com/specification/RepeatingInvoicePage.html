Feature: Creating New Repeating Invoice

Acceptance Criteria

  1. I can create a repeating invoice with an end date.
  2. I can create a repeating invoice without an end date.

Example 1: I can create a repeating invoice with an end date.

When:
I create a new repeating invoice with an end date 20 Oct 2020
Then:
It will create a repeating invoice with the end date 20 Oct 2020

Example 2: I can create a repeating invoice without an end date.

When:
I create a new repeating invoice without an end date
Then:
It will create a repeating invoice with no end date.
